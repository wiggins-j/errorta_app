# Errorta Homebrew tap

This repository (`errorta/homebrew-tap`) is an **install pointer** for the
`errorta` CLI. It contains exactly one file — `Formula/errorta.rb` — telling
Homebrew which pre-built release binary to download and its `sha256`.

**There is no product code here, ever.** The CLI source lives in
[`wiggins-j/errorta_app`](https://github.com/wiggins-j/errorta_app); the release
binaries are hosted as GitHub Releases on that repo.

## Install

```sh
brew install errorta/tap/errorta
# or:
brew tap errorta/tap && brew install errorta
errorta --help
```

The binary is fully self-contained (it embeds its sidecar + the AIAR engine), so
there is no Python or other runtime to install. First-run onboarding guides you
through `errorta connect …`.

> **Note:** the download is large (~100–200 MB) because the engine is bundled.

## How the formula is maintained

Do **not** hand-edit `Formula/errorta.rb`. It is generated by
`scripts/release-cli.sh` in the `errorta_app` repo, which — per platform —
builds + notarizes the binary, uploads it to the `errorta_app` GitHub Release,
and rewrites this formula with the new version + per-arch `url` + `sha256`, then
commits and pushes here. See `docs/BUILD_AND_RELEASE.md` in `errorta_app` for the
runbook.

## Status

Pre-1.0 / **alpha**. The CLI and its distribution are still stabilizing; expect
breaking changes between versions. Not yet in homebrew-core (that requires
homebrew-core acceptance — revisit post-1.0).
